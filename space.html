THE SPACE ODYSSEY

2020


The very first coding project we had on our first semester of the
Multimedia Design program at Zealand. The purpose of this project
was to create a concept and a website for a fictitious company that
is organizing trips to outer space. We were working in a group of
four using the basics of coding we learned in a week.


Role: designer and coder of the "Trip" page (see the 3rd picture).
